# tfdocker-example
Ubuntu 24-04 LTS docker container example with tensorflow and CUDA

# Build
Da shell, esegui
```
./build
```
Una volta terminato il processo di build, puoi eseguire il container.

# Esecuzione
Da shell, esegui
```
./run
```
